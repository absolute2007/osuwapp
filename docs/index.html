Небольшое Windows-приложение для отслеживания live PP, выбранной карты, последних игр и настраиваемого overlay поверх osu!stable.

Что делает приложение

Osuwapp читает данные из osu!.exe, показывает текущий контекст карты и сохраняет историю сессии между запусками.

Live PP

Текущий PP, accuracy, combo, hit counts и разбор по компонентам.

Beatmap preview

Предпросмотр PP для выбранной карты в song select с учетом модов.

Recent plays

Список последних результатов, который сохраняется после закрытия приложения.

In-game overlay

Настраиваемые блоки PP, статистики, hit counts и текста карты поверх игры.

Скриншоты

Несколько текущих визуальных артефактов проекта: интерфейс, overlay и иконка.
